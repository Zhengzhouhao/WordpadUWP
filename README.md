# WordpadUWP
A UWP version of... Wordpad!

This app features a simple and easy to use interface other than ribbons

**Features:**

  • Bold/Italic/Underline/Strikethough
  
  • Superscript/Subscript
  
  • Change font size
  
  • Change font color
  
  • Change font family
  
  • Align to left/center/right
  
  • Add image/hyperlink
  
  • Find text
  
  • Undo/redo
  
**Planned features:**

  • Replace selected text
